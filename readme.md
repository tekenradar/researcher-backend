# Researcher Backend

- `GIN_DEBUG_MODE`

- `RESEARCHER_BACKEND_LISTEN_PORT`
- `CORS_ALLOW_ORIGINS`
- `USE_DUMMY_LOGIN`
- `LOGIN_SUCCESS_REDIRECT_URL`
- `API_KEYS`

- `SAML_IDP_URL`

- `SAML_ENTITY_ID`
- `SAML_IDP_METADATA_URL`
- `SAML_SESSION_CERT_PATH`

- `RESEARCHER_DB_CONNECTION_STR`

- `RESEARCHER_DB_CONNECTION_PREFIX`
- `DB_TIMEOUT`

- `ADDR_STUDY_SERVICE`
- `ADDR_EMAIL_CLIENT_SERVICE`
- `GRPC_MAX_MSG_SIZE`
